# Project Title: Game Framework

A simple 2d platformer to demonstrate collision and movement

## Instructions for Build and Use

Steps to build and/or run the software:

1. Create your own art assests
2. Code in Gdscript
3. Use some presets to help

Instructions for using the software:

1. Boot up Godot 
2. Open up the project from github
3. simply run the project using the play button

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Godot
* Krita (Art Program)
* Github
